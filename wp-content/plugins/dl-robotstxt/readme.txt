== Description ==

DL Robots.txt Позволяет редактировать содержимое вашего файла robots.txt и задавать оптимальные настройки для поисковых одним кликом мышки

Если вы хотите помочь в разработке этого плагина - добро пожаловать на [github](https://github.com/dyadyal/dl-robots-txt)

== Installation ==

1. Распакуйте архив с плагином
2. Загрузите папку с плагином в директорию /wp-content/plugins/ на вашем сервере
3. Активируйте плагин в разделе Плагины в панели администрирования WordPress
4. Перейдите на страницу на страницу настроек
5. Внесите неоходимые изминения или выберите оптимальные настройки
6. Сохраняем изминения
