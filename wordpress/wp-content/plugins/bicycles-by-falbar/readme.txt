== Description ==

Collection of ready-made solutions for WordPress customization.

It can:

* Remove meta generator;
* Remove RSD link;
* Remove WLW manifest link;
* Remove next and prev links;
* Remove canonical link;
* Remove shortlink link;
* Remove dns-prefetch links;
* Disable wp-embed;
* Disable emoji;
* Automatically set the alt attribute;
* Remove .recentcomments styles;
* Remove attachment pages;
* Remove archives date;
* Remove archives tag;
* Remove post pagination;
* Remove archives author;
* Hide login errors;
* Disable XML-RPC;
* Disable X-Pingback;
* Enable filename lowercase;
* Disable RSS feeds;
* Disable REST API;
* Remove links in admin bar;
* Remove jQuery Migrate;
* Remove Querystrings;
* Replace external links in comments on the JavaScript code;
* Replace external links from comment authors on the JavaScript code;
* Disable Embeds;
* Disable Emojis;
* Remove field "Website" from comment form;
* And many other functions.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New';
2. Search for 'Bicycles by falbar';
3. Activate Bicycles by falbar from your Plugins page.

= From WordPress.org =

1. Download Bicycles by falbar;
2. Upload the 'Bicycles by falbar' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...);
3. Activate Bicycles by falbar from your Plugins page.
